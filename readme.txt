Procédure pour les tests:
    - tar -xf Projet-mirzazad.tar.gz
    - make
    - ./testing.sh > codes.txt

Le script testing.sh lance les tests un par un.
Pour générer un code backend a partir d'un fichier frontend particulier
Il faut lancer la commande: ./frontend fichier_src fichier_dst

Tous les tests fonctionnent sauf functions.c et listes.c
